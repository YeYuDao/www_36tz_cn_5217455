<br/></br>### 课程介绍:<br/></br>JavaScript在早期的设计中就没有模块、包、类的概念，开发者需要模拟出类似的功能，来隔离、组织复杂的JavaScript代码，我们称为模块化。
模块就是一个实现特定功能的文件，有了模块我们就可以更方便的使用别人的代码，要用什么功能就加载什么模块。
JS 前端模块化的发展历史， 优缺点及应用（IIFE， AMD， CMD， ES6）， 包含一个模块化案例贪吃蛇。
